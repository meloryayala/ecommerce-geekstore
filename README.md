## 📋 Requirements

**-Entities:**
* Products
* Categories
* Users
* Cart products
      

**-Features:**
* User login/ logout 🦱
* Display products by category 🏷️
* Add products to Cart, increase quantity and delete 🗑️ 🛒
* Payment integration with Strapi 💰
* Manage all  purchases 📦
